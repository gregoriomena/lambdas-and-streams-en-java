1.1 Novedades de Java 8

  • Interfaces funcionales. Las interfaces ahora van a permitir cierta implementación de código por defecto. También permite la herencia múltiple
  • Expresiones Lambda. Va a permitir tener código más elegante así como hacer uso de la interfaces funcionales de una forma sencilla
  • Métodos referenciados. Permiten referenciar funciones por sus nombres, usando funciones como si fueran parámetros. Están relacionados con las expresiones lambdas
  • Métodos default en las interfaces
  • API Stream. Ofrece una manera revolucionara de tratar con las colecciones. Se complementa con las expresiones lambda
  • Clases Optional, para manejar propiedades con valores NULL
